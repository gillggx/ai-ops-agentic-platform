圖表樣式 Agent 化 — 11 種圖前後對照

每種圖左邊是現況 agent 能畫出的極限、右邊是補完參數後同一句指令該得到的圖， 下方附 agent 該下的參數（粗體 = 新增）。所有圖可互動 — 滑鼠移到資料點上感受 tooltip 差異。 模擬資料：EQP-01 STEP_001 xbar 序列（含 R1 突破與 R2 連續同側）。

01line_chart 趨勢圖 —「標準 SPC 管制圖，提示顯示 lotID」

現況三條線 + 紅圈；提示只有 x/value
沒有 σ 區帶、軸沒名字、紅圈不解釋原因。
期望σ 區帶 + 可解釋違規 + 指定欄位提示
Zone C/B/A、CL 實線、UCL/LCL 紅虛線帶值、滑過紅點顯示違規規則 + lotID/recipe、y 軸有單位。
{ "x":"eventTime", "y":"value", "ucl_column":"ucl", "lcl_column":"lcl", "center_column":"center",
  "highlight_field":"is_ooc", "style":{"spc_zones":true,"y_label":"xbar (nm)"},
  "tooltip_fields":["lotID","recipe"], "weco_annotate":true }

S5 反向題 —「簡潔版，不要區帶」

期望agent 下 style.spc_zones=false
管制線與提示照給，只拿掉區帶 — 測 agent 聽懂指令 vs 無腦全開。
S5 是關鍵題：文件必須寫成原則（「spc_zones 控制 σ 區帶顯示」），而不是 case 規則（「標準=開」）， agent 才能雙向操作 — 這正是 STYLE-10 要驗的「有沒有看懂 document」。

02xbar_r X̄-R 管制圖 —「違規點要能看出違反哪條規則」

現況WECO 紅點會標，滑過毫無反應
這張圖完全沒有 tooltip（試滑 — 沒有任何提示）。
期望σ 區帶（本來就算好）+ 違規原因進提示
滑過紅點顯示「違反 R2：連續 9 點同側」等可行動資訊。
{ "value_column":"value", "weco_rules":["R1","R2","R3"],
  "weco_annotate":true, "style":{"spc_zones":true}, "tooltip_fields":["subgroup","n"] }

03imr I-MR 管制圖 —「單值管制圖，含區帶與違規說明」

現況I 圖裸線 + 紅點，無 tooltip、無區帶
與 xbar_r 同病：紅點知其然不知其所以然。
期望區帶 + 違規提示（與 xbar_r 同構）
I 圖σ 由移動全距推（MR̄/d2），區帶零額外計算成本。
{ "value_column":"value", "weco_rules":["R1","R2"],
  "weco_annotate":true, "style":{"spc_zones":true} }

04spc_panel 複合面板 —「三站 xbar 面板，每格標準樣式」

現況每格裸趨勢線，子圖樣式不可控
面板只能選 chart_name / event_filter，style 進不去子圖。
期望style / tooltip_fields 透傳到每格
三格都有區帶與管制線，滑點都有 lotID 提示 — 一個參數管全面板。
{ "step":["STEP_001","STEP_002","STEP_003"], "chart_name":"xbar_chart",
  "style":{"spc_zones":true}, "tooltip_fields":["lotID"] }

05histogram_chart 直方圖 —「加規格線 LSL 48 / USL 52，疊常態曲線」

現況裸柱，無規格線、無提示
Cpk 討論必看「分佈壓不壓得進規格」— 現在畫不出來。
期望LSL/USL + 常態疊線 + bin 提示
滑過每根柱顯示區間與筆數。
{ "value_column":"value", "bins":18, "lsl":48, "usl":52, "normal_overlay":true }

06probability_plot Q-Q 圖 —「常態性檢定，含信賴帶」

現況點 + 參考線，無信賴帶、無提示
偏離點是哪筆資料？現在滑過去不知道。
期望95% 信賴帶 + 點提示（值/分位）
帶外點一眼可見，滑過顯示實際值與理論分位。
{ "value_column":"value", "confidence_band":true, "tooltip_fields":["lotID"] }

07box_plot 盒鬚圖 —「各 lot 分佈，疊原始點，提示顯示樣本數」

現況盒體 + 離群點，提示內容寫死
看不到分佈內部的原始資料密度。
期望show_points 原始點疊圖 + n 進提示
jitter 原始點露出真實密度，滑盒體顯示五數摘要 + 樣本數。
{ "y_label":"xbar", "show_outliers":true, "show_points":true, "tooltip_fields":["n"] }

08scatter_chart 散佈圖 —「相關性分析，加趨勢線」

現況裸點；提示只有 x/y 值
相關方向與強度要用眼睛猜。
期望trend_line 回歸線 + lotID 提示
趨勢線 + R² 標註，滑點看該筆 lot。
{ "x":"pressure", "y":"etch_rate", "trend_line":true, "tooltip_fields":["lotID"] }

09bar_chart 長條圖 —「由多到少，柱上標數值」

現況排序可以，數值要滑過才看得到
期望show_values 一眼讀值 + y 軸名
{ "x":"toolID", "y":"ooc_count", "order":"desc", "show_values":true, "style":{"y_label":"OOC 次數"} }

10pareto 柏拉圖 —「標 80% 線與累積百分比」

現況柱 + 累積線 + 80% 門檻線（已可畫）
累積線上沒有 % 數字，要滑過才知道累積到多少。
期望show_cum_labels 累積 % 直接標
「前兩類佔 72%」這種結論不用滑鼠就能唸出來。
{ "category_column":"chart_name", "cumulative_threshold":0.8, "show_cum_labels":true }

11ewma_cusum 漂移偵測 —「越界點標註、界限帶上色」

現況CUSUM 線 + h 界限線
越界從哪一點開始？要自己對。
期望annotate_crossings 越界點紅標 + 界限帶
±h 帶淡紅、首個越界點紅點 + 提示「CUSUM 4.6 > h=4，漂移始於此」。
{ "mode":"cusum", "k":0.5, "h":4, "annotate_crossings":true, "style":{"h_band":true} }
