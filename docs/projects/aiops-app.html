aiops-app

Next.js 15 standalone 前端。只做 UI 渲染 + 走 /api/* proxy 把請求轉到 Java / Sidecar。沒業務邏輯、不直連 DB / simulator。

Port8000
TechNext.js 15 (App Router) · React 19 · Node 20.18 · NextAuth v5
Service unitaiops-app.service
Deploydeploy/update.sh (npm build + standalone copy + restart)

主要頁面

/admin/pipeline-builder/*

Glass Box 建 pipeline 主介面。Plan card → confirm → agent 動 canvas → save。

/admin/skill-library

Skill 編輯 + 綁定 pipeline + 設 trigger_config (schedule / event / user)。

/admin/build-traces

看歷史 build trace。Summary tab 走 trace_summary 模組合成 plan/phase/attempt 結構化視圖。Journey / Results / Pipeline tabs 看 raw。

/admin/block-docs

編 block 文檔的 Markdown body。寫入 block_docs DB。sidecar inspect_block_doc + advisor_v2 都從這拿內容。

/admin/users

使用者 + 角色管理 (IT_ADMIN / PE / ON_DUTY)。role_change_logs 全紀錄。

/alarms

Alarm panel: 列表、ack、dispose。Severity filter + 時間排序。

/chat

Chat panel — 走 sidecar agent_orchestrator_v2。intent_classifier 5-bucket 路由。

關鍵 UI components

AgentBuilderPanelV30

處理 SSE event stream: goal_plan_proposed / phase_round / phase_action / phase_completed / build_finalized. handleEvent dispatcher 約 144-302 行。

Summary tab (v6.2 新加) call /summary endpoint 拿結構化 JSON, 渲染 plan card + per-phase attempt 詳情。Journey/Results/Pipeline 仍 raw view。

BlockAdvisorPanel

右側問答區

顯示 advisor_progress chips (LLM 的 tool calls) + advisor_answer markdown. UX 跟 legacy 一致, 後端換成 tool-loop。

ChartRenderer (charts/*)

18 個 SVG chart components

line_chart / bar_chart / box_plot / probability_plot / ewma_cusum / xbar_r / imr / pareto / splom / ... 每個是獨立 React component, 從 chart_spec.snapshot 渲染。

Standalone build gotcha

Next.js standalone mode: npm run build 單獨跑會留下 broken .next/static 路徑。必須 cp -r .next/static .next/standalone/.next/ + cp -r public .next/standalone/。deploy/update.sh 已處理好。
