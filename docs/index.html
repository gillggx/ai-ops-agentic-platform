What's new in v6.3 (2026-05-21)

  • Verifier 改 deterministic structural-only check — judge / rows-gate / deficit / empty_data 全砍, 移到 runtime concern (v30.20)
  • Agent-driven verify trigger — verifier 不再每個 mutation 自動跑, agent 主動 call run_verifier (v30.22)
  • Covers gate 變 feature flag default OFF — phase.expected 退為 hint, 不擋 advance (v30.23)
  • Block doc digest 灌全文 (~2.5K chars) — agent 看到 admin 編的完整 Markdown, 不再只看 param schema (v30.21)
  • Multi-tool-use strip — LLM 一次 emit 多個 tool_use 不會再 400 error
  • Stringified params coercer — agent 寫 params="{...}" 自動 json.loads
  • Skill auto-trigger 完工 (v6.1) — java-scheduler 認 skill_documents.trigger_config, alarm 流恢復
  • Tool-using doc Q&A agent (v6.2) — Advisor 換成 Anthropic tool-loop, admin-edited Markdown 直接進答案
  • block_groupby_agg param rename — agg_func/agg_column → aggregate/column (對齊 step_check)
  • block_ewma_cusum target=null 支援 — schema 允許 null, executor 自動 mean(value_column)
