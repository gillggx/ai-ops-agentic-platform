4. compile_chunk 內的 12 種 deterministic autofix

每次 LLM 產 ops 後、validate 前跑（按順序）：

  1. _auto_insert_unnest — filter 引用 nested leaf 時自動補 block_unnest
  2. _auto_rewire_chart_chains — 接錯 chart_spec→data 改接 dataframe upstream
  3. _autocorrect_filter_values — fuzzy match (例 'xbar' → 'xbar_chart')
  4. _autostrip_singleton_groupby — 高 cardinality group_by 收成 n=1 時自動 strip
  5. _normalize_select_fields — string array → [{path:x}]
  6. _normalize_set_param_ops — malformed set_param 折回 add_node
  7. _drop_unspecced_cpk — block_cpk 沒給 usl/lsl 就 drop
  8. _force_skill_terminal (v18) — skill_step_mode 最後一步用 block_threshold → 改 block_step_check
  9. _autocorrect_column_refs (v18) — LLM 亂編 column 名時 fuzzy 配 upstream
  10. _drop_malformed_ops — None id 等垃圾 op 丟掉
  11. _resolve_input_refs — $X 沒宣告時用 instruction 裡的 [BRACKET] 補
  12. _dedup_against_plan — 跟既有 plan 重複的 add_node 丟掉
