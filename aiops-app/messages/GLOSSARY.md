# 翻譯 Glossary（4 語系）

翻譯任何 UI 字串前必讀。規則：
1. 專有名詞/縮寫（OOC、Cpk、Cp、xbar、UCL/LCL/CL、WECO、SPC、APC、RAG、SSE、pipeline、block、phase、round、Skill、Builder、Planner、Repair、Verifier、W1/W2/W3、node/edge）**四語系一律保留英文原文**。
2. 日文採です・ます体（UI 按鈕/標籤可用名詞止め）。
3. 簡中不是繁中的字轉換 — 按下表術語覆寫。
4. 禁 emoji；幾何符號（✓ ✕ ▲ ◆ ○ ● ▣ ◈ ▸）視為圖示，不翻譯。
5. `{param}` ICU 參數與 mono 格式（r{n}/32）原樣保留。
